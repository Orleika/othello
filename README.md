# ファジィオセロ
ファジィ理論を用いたオセロAI

コンパイル
```
javac -encoding UTF-8 *.java
```

使用法
```
# COM vs COM
java Server
java AI1 ${hostname} <white/black>
java AI1 ${hostname} <white/black>

# COM vs HUMAN
java Server
java AI1 ${hostname} <white/black>
java Controller ${hostname} <white/black>
```
